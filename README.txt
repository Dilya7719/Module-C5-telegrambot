В коде используется API https://www.exchangerate-api.com/docs/pair-conversion-requests
Зарегистрируйтесь на сайте и получите свободный ключ ("Get Free Key")
Ключ запишите в файл "config.py" вместо "API_ключ"

Токен бота также укажите свой в файл "config.py" вместо "ТОКЕН_бота"

Для запуска бота - запустите файл bot_script.py
